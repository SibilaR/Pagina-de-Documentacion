Introduccion al HTML

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y estructurar contenido en la web.

Permite definir la estructura y el significado de los elementos dentro de una página web mediante etiquetas. Estas etiquetas permiten a los desarrolladores web organizar el contenido en títulos, párrafos, listas, imágenes, enlaces, formularios y mucho más.

Estructura Básica de un Documento HTML

Un documento HTML consta principalmente de dos secciones: el encabezado (< head>) y el cuerpo (< body>). 


    
< head>
    < meta charset="UTF-8">
    < meta name="viewport" content="width=device-width, initial-scale=1.0">
    < title>Título de la Página< /title>

< body>
    < h1>¡Hola, mundo!< /h1>
    < p>Este es un párrafo de ejemplo.< /p>
< /body >

El encabezado contiene metadatos, como el título de la página, enlaces a hojas de estilo CSS y scripts JavaScript.

El cuerpo contiene el contenido visible de la página, definido mediante etiquetas HTML.

Elementos HTML Fundamentales

  • Encabezados (< h1 >, < h2 >, < h3 >, < h4 >, < h5 >, < h6 >): Definen títulos y subtitulares.
  • Párrafos (< p >): Utilizados para texto normal.
  • Listas (< ul >, < ol >, < li >): Para crear listas no ordenadas y ordenadas.
  • Enlaces (< a >): Permiten enlazar a otras páginas web o recursos
  • Imágenes (< img >): Para mostrar imágenes en la página.

Atributos en HTML

Las etiquetas HTML pueden tener atributos que proporcionan información adicional sobre el elemento. Por ejemplo, el atributo src en la etiqueta < img > especifica la ruta de la imagen que se mostrará.

< img src="imagen.jpg" alt="Descripción de la imagen" >

Formularios en HTML

Los formularios (< form >) permiten a los usuarios enviar datos a un servidor web.

Contienen varios tipos de controles, como campos de texto (< input type="text">), casillas de verificación (< input type="checkbox" >), botones (< button >), etc. Los datos introducidos por el usuario pueden ser procesados por el servidor. 


< form action="/procesar" method="post" >
    < label for="nombre"> Nombre: 
    < input type="text" id="nombre" name="nombre" >
    < button type="submit"> Enviar 
< /form >
